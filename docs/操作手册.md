# 操作手册

## 1. 环境准备

- Python `>= 3.10`
- Node.js `>= 18`（仅前端构建需要）
- 建议在受信任内网运行 Dashboard

安装依赖：

```bash
python -m pip install -r requirements.txt
```

## 2. 快速启动（首次）

### 2.1 初始化小说项目

```bash
python -X utf8 webnovel-writer/scripts/webnovel.py init ./webnovel-project "你的书名" "题材"
```

### 2.2 绑定工作区

```bash
python -X utf8 webnovel-writer/scripts/webnovel.py use "<PROJECT_ROOT>" --workspace-root "<WORKSPACE_ROOT>"
```

### 2.3 预检

```bash
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT_OR_WORKSPACE_ROOT>" preflight --format json
```

## 3. 日常命令

```bash
python -X utf8 webnovel-writer/scripts/webnovel.py where
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT>" extract-context --chapter 1 --format text
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT>" status -- --focus all
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT>" index -- stats
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT>" rag -- stats
```

> `status/index/rag/workflow` 等透传命令，参数放在 `--` 之后。

## 4. Dashboard 启停

启动：

```bash
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT>" dashboard --host 127.0.0.1 --port 8765
```

常用参数：

- `--no-browser`：不自动打开浏览器
- `--no-bootstrap-index`：不自动初始化缺失索引

停止：前台进程按 `Ctrl+C`。

## 5. 功能操作 SOP

### 5.1 设定词典

1. 在 `Settings` 页面读取 `设定集` 文件树。
2. 触发词典抽离（`POST /api/settings/dictionary/extract`）。
3. 在冲突列表中人工决策（`POST /api/settings/dictionary/conflicts/{id}/resolve`）。
4. 确认词典落盘：`.webnovel/dictionaries/setting-dictionary.json`。

### 5.2 总纲拆分/重拆

1. 打开 `OutlineWorkspacePage` 同屏查看总纲/细纲。
2. 先 preview，再 apply：
   - `split/preview -> split/apply`
   - `resplit/preview -> resplit/apply`
3. 发生顺序冲突时先处理冲突，再重新 apply。
4. 检查落盘：`.webnovel/outlines/*` 与 `大纲/细纲.md`。

### 5.3 文本协助修改

1. 先调用 `POST /api/edit-assist/preview` 生成提案。
2. 作者确认后调用 `POST /api/edit-assist/apply`。
3. 失败时优先检查版本冲突（`409`），重新 preview 再 apply。
4. 检查日志：`.webnovel/edits/assist-log.jsonl`。

## 6. 迁移、备份与恢复

### 6.1 历史 `.claude` 指针迁移

```bash
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT_OR_WORKSPACE_ROOT>" migrate codex --dry-run
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT_OR_WORKSPACE_ROOT>" migrate codex
```

### 6.2 备份与归档

```bash
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT>" backup -- --list
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT>" backup -- --rollback 12
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT>" archive -- --stats
python -X utf8 webnovel-writer/scripts/webnovel.py --project-root "<PROJECT_ROOT>" archive -- --restore-character "<角色名>"
```

## 7. 常见问题排查

| 现象 | 排查建议 |
|---|---|
| `403 workspace_forbidden / workspace_mismatch` | 检查 `workspace_id` 与 `project_root` 是否属于同一工作区 |
| `409 OUTLINE_*_LOCK_TIMEOUT` | 有并发拆分/重拆写入，稍后重试 |
| `409 EDIT_ASSIST_*_VERSION_CONFLICT` | 文件已变化，重新 preview 后再 apply |
| `501 EDIT_ASSIST_UNAVAILABLE` | 外部编辑能力不可用，稍后重试或切换模式 |
| 页面出现 mock 数据 | 检查前端是否启用了 mock fallback |

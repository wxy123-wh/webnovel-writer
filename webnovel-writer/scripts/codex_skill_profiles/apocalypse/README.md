# Apocalypse Skill Profile — 末世生存

> **Profile ID**: `apocalypse`  
> **适用题材**: 末世生存、丧尸围城、废土求生、灾变进化等以极端环境下的生存为核心的网文

## 概述

Apocalypse Profile 为末世生存题材提供专项写作辅助。末世文的核心张力来自"文明崩塌后的重建与抉择"——在极端环境下，人性的光辉与阴暗同时被放大到极致。

本 profile 聚焦末世生存的八大核心要素：

1. **生存压力** — 资源永远不够的紧迫感
2. **道德拷问** — 人性在极端环境下的两难抉择
3. **人性展现** — 不同角色的差异化反应
4. **基地建设** — 防御→扩张→危机的循环
5. **能力进化** — 有代价的超自然成长
6. **团队配置** — 互补的核心团队
7. **末世规则** — 世界设定前后一致
8. **希望维护** — 黑暗中的一线光明

## 文件结构

```
apocalypse/

├── rules.md            # 末世生存专项写作规则（8 条核心约束）
└── check-weights.yaml  # Checker 权重覆盖配置
```

## 加载规则

- **会话级加载**: 通过 `codex session start --profile apocalypse` 启动时加载
- **推荐叠加**: `--profile apocalypse,battle` (战斗场景) 或 `--profile apocalypse,consistency` (世界规则校验)
- **跨题材适配**: 可与 xianxia/urban 等叠加（如末世修仙、末世异能）

## Checker 权重说明

| Checker | 权重 | 说明 |
|---------|------|------|
| pacing | **1.2** | 末世文节奏偏紧张，停滞阈值收紧至 3 章 |
| coolpoint | **1.3** | 生存突破/打怪是爽点来源，每章至少 1 个 |
| consistency | **1.1** | 末世规则对一致性要求高 |
| hook | **1.4** | 章末悬念在末世文中尤为关键 |

## 题材特点速查

- 生存压力是底色，资源永远紧缺
- 人性考验是核心深度来源
- 基地建设线提供安全感和成长感
- 即使最黑暗也要留一线希望
- 适合与多种题材融合（末世修仙、末世系统、末世重生等）

# 末世生存写作规则 (Apocalypse Writing Rules)

> 以下规则由 `apocalypse` profile 加载，适用于末世生存、丧尸围城、废土求生等以极端环境为核心的题材。

---

## 规则 A-01：生存压力 — 资源紧缺

**约束**: 核心生存资源（食物/水/弹药/药品/能源）必须 **持续紧缺**，不能出现长期的富裕状态：

### 资源紧缺设计

| 资源类型 | 紧缺表现 | 恢复方式 |
|---------|---------|---------|
| **食物/水** | 始终在"够用3天"的水平 | 搜索/种植/交易（但不能一劳永逸） |
| **弹药** | 每次战斗都在消耗库存 | 搜索军械库/自制/交易 |
| **药品** | 受伤后才发现不够用 | 搜索医院/自制草药/交换 |
| **能源** | 夜晚不能随意用灯/设备 | 太阳能/发电机燃料/搜刮电池 |
| **信息** | 对外界状况一无所知 | 电台/幸存者口述/无人机 |

### 紧张感维持法则

1. **刚够法则**: 资源获取量始终"刚够"或"差一点"，从不宽裕
2. **消耗可见**: 每次使用资源都要提及存量的变化（"这是倒数第三瓶水了"）
3. **获取代价**: 每次获取资源都伴随风险（搜索=危险、交易=妥协）
4. **新需求**: 每次解决一个资源问题，新的需求就会产生（修好了发电机但需要燃料）

**违反示例**: 第 5 章找到一个大仓库，食物够吃半年，之后 30 章再无资源压力。

**合规示例**: 第 5 章找到仓库，但库存只够 2 周；运回途中遇到丧尸群损失了一半；到达基地后发现其他幸存者也需要食物，不得不分配——实际到手只够 5 天。

## 规则 A-02：道德拷问 — 人性两难

**约束**: 每 **5-10 章** 设置 **1 个道德两难选择**，没有标准答案：

### 两难设计模板

| 类型 | 两难描述 | 示例 |
|------|---------|------|
| **救人困境** | 救一个人但危及团队 vs 保团队但放弃一人 | 被困的陌生人可能引来丧尸群 |
| **资源分配** | 公平分配 vs 效率优先 | 药品给受伤者还是给战斗主力 |
| **信任危机** | 信任新来者 vs 保护现有团队 | 新幸存者可能是间谍 |
| **手段选择** | 人道手段 vs 有效手段 | 用活人做诱饵引开丧尸 |
| **理想现实** | 坚持原则 vs 生存需要 | 不杀无辜者 vs 对方会暴露基地位置 |
| **短期长期** | 即时利益 vs 长远发展 | 抢夺邻队资源 vs 建立联盟 |

### 两难写作原则

1. **两方都有理**: 不能让读者轻易判断"应该选A"，两边都要有充分的理由
2. **有后果**: 选择后必须展示真实后果（不能选完就忘了）
3. **角色分歧**: 团队内部对选择产生分歧，展现不同价值观
4. **影响关系**: 道德选择应该影响角色之间的关系和信任

**违反示例**: 团队一致同意救陌生人，没有任何人反对，救人过程顺利无代价。

**合规示例**: 救还是不救的争论持续了整整一章，队长最终决定救但有人因此离开团队，被救者后来确实带来了麻烦但也带来了关键信息——选择的后果是混合的。

## 规则 A-03：人性展现 — 差异化反应

**约束**: 不同角色面对同一危机时必须展现 **不同反应**，避免全员英雄或全员反派：

### 反应光谱

```
恐惧退缩 ←——————→ 勇敢担当
自私自保 ←——————→ 舍己为人
理性冷静 ←——————→ 情绪崩溃
信任他人 ←——————→ 猜疑防备
服从规则 ←——————→ 叛逆挑战
```

每个角色在光谱上占据不同位置，且同一角色在不同类型危机中位置可能不同。

### 角色反应设计

1. **核心恐惧**: 每个角色有一个核心恐惧（怕死/怕失去/怕被遗弃/怕变成怪物）
2. **应激模式**: 面对压力时的本能反应（战斗/逃跑/僵住/讨好）
3. **成长方向**: 角色在末世中会朝哪个方向变化（变强/变疯/变善良/变冷酷）
4. **崩溃点**: 每个角色有一个"承受上限"，超过会崩溃或转变

**违反示例**: 团队 5 人面对丧尸群，全部冷静应对、配合默契、无一畏惧。

**合规示例**: 队长冷静指挥（理性型）、老兵冲在最前面（战斗型）、新成员吓得不敢动（僵住型）、医疗兵边哭边给伤员包扎（情绪化但尽责）、智囊在找逃跑路线（回避型）。

## 规则 A-04：基地建设 — 防御→扩张→危机循环

**约束**: 如有基地/避难所线，需遵循 **防御→扩张→危机** 的循环模式：

### 循环结构

```
[防御阶段] → [扩张阶段] → [危机阶段] → [重建防御] → ...
```

| 阶段 | 内容 | 建议篇幅 |
|------|------|---------|
| **防御** | 建立基本防御、储备资源、制定规则 | 3-5 章 |
| **扩张** | 招人、扩大领地、建立产能 | 3-5 章 |
| **危机** | 外敌入侵/内部叛乱/资源危机/丧尸潮 | 3-8 章 |
| **重建** | 从危机中恢复，比之前更强 | 2-3 章 |

### 基地建设要素

1. **物理防御**: 墙壁/陷阱/瞭望塔/武器储备
2. **生活保障**: 食物来源/水源/医疗/住所
3. **管理制度**: 领导机制/资源分配/惩罚规则
4. **人口管理**: 招人标准/人员分工/内部矛盾处理
5. **外部关系**: 与其他幸存者团体的关系（合作/竞争/敌对）

**违反示例**: 基地建成后一劳永逸，再无安全问题，后续只专注于探索。

**合规示例**: 第一轮：木栅栏防御 → 搜刮扩容 → 小规模丧尸潮 → 升级为铁墙 → 第二轮：更多人口 → 建立农场 → 人类敌对势力来袭 → 升级防御体系 → 第三轮……

## 规则 A-05：能力进化 — 有代价的超自然成长

**约束**: 如有异能/进化设定，进化必须有 **明确的代价**：

### 代价类型

| 代价类型 | 说明 | 表现形式 |
|---------|------|---------|
| **身体负荷** | 进化对身体的损伤 | 变异组织/器官压力/身体失控 |
| **精神影响** | 进化对精神的影响 | 记忆丢失/人格变化/幻觉 |
| **资源消耗** | 进化需要的额外资源 | 需要大量食物/特殊晶体/能量 |
| **人性侵蚀** | 越强越不像人 | 外貌变化/情感减弱/本能增强 |
| **社会隔阂** | 进化者与非进化者的差距 | 被恐惧/被利用/被排斥 |

### 进化设计原则

1. **渐进式**: 进化不是一次性完成，而是分阶段，每阶段有不同代价
2. **选择性**: 某些进化路径需要主动选择，选了就不能回头
3. **不可逆**: 已经付出的代价不能撤销（增加了选择的重量）
4. **风险递增**: 越高级的进化，失败的代价越大（死亡/失控/变成怪物）

**违反示例**: 主角每次进化都变强，没有任何副作用，且可以无限进化。

**合规示例**: 主角进化后获得超强力量，但需要消耗大量食物（一天吃十人份），且过度使用时手臂会不受控制地异变。到后期，他不得不在"继续变强但失去人性"和"保持人性但战力不足"之间抉择。

## 规则 A-06：团队配置 — 互补核心团队

**约束**: 核心团队 **3-5 人**，各有所长，能力互补：

### 团队角色模板

| 角色 | 功能 | 必备特质 |
|------|------|---------|
| **领导者** | 决策、凝聚团队 | 决断力、责任感 |
| **战斗核心** | 正面作战 | 强战斗力、勇敢 |
| **技术/智囊** | 解决技术问题、制定计划 | 知识面广、冷静 |
| **医疗/后勤** | 治疗伤员、管理资源 | 细心、有同情心 |
| **斥候/联络** | 侦察、对外沟通 | 敏捷、善于社交 |

### 团队动态原则

1. **互补非重叠**: 每个人有独特的不可替代的价值
2. **内部矛盾**: 团队内部有合理的分歧和冲突（不是所有人都服气队长）
3. **有死有伤**: 核心团队不是全员安全，适当的牺牲增加紧张感
4. **动态变化**: 团队成员可以加入、离开、牺牲，不是固定不变的
5. **每个人都有故事**: 不是工具人，每个团队成员都有自己的背景和动机

**违反示例**: 5 人团队全是近战型，能力重叠，彼此毫无矛盾，每个人都愿意为主角赴死。

**合规示例**: 队长擅长决策但体力一般；狙击手战力最强但性格孤僻；工程师负责技术但怕血；医生治疗能力强但不善战斗；斥候速度快但曾经是敌人那边的人，信任危机始终存在。

## 规则 A-07：末世规则 — 世界设定一致性

**约束**: 末世世界的核心规则（丧尸特性/灾变机制/异能来源）必须 **前后一致**：

### 必须明确的世界规则

| 规则类别 | 需要回答的问题 |
|---------|--------------|
| **灾变起源** | 怎么发生的？为什么发生？能恢复吗？ |
| **丧尸/怪物** | 什么触发？弱点是什么？会进化吗？怎么传播？ |
| **异能/进化** | 谁能获得？怎么获得？上限在哪？副作用？ |
| **社会结构** | 政府还在吗？军方呢？谁掌握权力？ |
| **自然环境** | 天气/气候受影响了吗？动物呢？植物呢？ |
| **时间线** | 灾变多久了？社会退化到什么程度？ |

### 一致性保障

1. **规则前宣**: 核心规则在早期（前 10 章）就应建立
2. **不可矛盾**: 一旦确立的规则不可被后续情节打破
3. **逻辑推演**: 规则的后果要推演到位（如果丧尸怕火，那人类聚集点必然大量用火）
4. **渐进揭示**: 细节可以逐步补充，但核心规则不可后期修改

**违反示例**: 第 5 章说丧尸怕火，第 30 章出现不怕火的丧尸变种但无任何解释。

**合规示例**: 第 5 章确认普通丧尸怕火 → 第 25 章首次出现火抗丧尸 → 通过研究发现是进化现象 → 此后战斗策略必须调整。

## 规则 A-08：希望维护 — 黑暗中的光

**约束**: 即使是最黑暗的场景，也必须留有 **一线希望**：

### 希望的表现形式

| 场景 | 黑暗元素 | 必须保留的希望元素 |
|------|---------|-------------------|
| **战斗失败** | 团队伤亡惨重 | 有人活下来了/发现了敌人的弱点 |
| **基地沦陷** | 失去家园 | 带出了关键资源/人员的羁绊更深了 |
| **人性最暗** | 人类互相背叛 | 有人选择信任/有人挺身而出 |
| **个人绝境** | 被困/受伤/孤立 | 一线生机/意外的救援信号 |
| **真相揭示** | 灾变不可逆转 | 人类可以适应/新世界的可能性 |

### 希望设计原则

1. **不撒谎**: 希望不能是虚假的（不能骗读者说一切会好，然后更糟）
2. **微小但真实**: 希望不需要宏大（一个孩子的笑容、一株新芽、一段善意的对话）
3. **代价换取**: 最黑暗时刻的希望，往往来自某个角色的牺牲或选择
4. **循环存在**: 希望→绝望→新希望 的循环，每次都更深沉

**违反示例**: 连续 10 章全是黑暗、绝望、死亡，没有任何积极的元素，读者情绪被完全压垮。

**合规示例**: 基地沦陷、队长牺牲——但在废墟中，一个小女孩递给主角一块糖，说"大叔别哭"。这块糖不能拯救世界，但它证明这个世界还值得拯救。

# 描写场景写作规则 (Description Scene Rules)

> 以下规则由 `description` profile 加载，适用于场景描写、人物出场、氛围营造等非动作密集型内容。

---

## 规则 D-01：五感法则 — 多感官描写

**约束**: 每个重要场景（超过 500 字的环境描写段落）至少覆盖 **3 种感官** 描写：

| 感官 | 常见表现 | 典型用法 |
|------|---------|---------|
| **视觉** | 色彩、光影、形状、动态 | 基础场景建构 |
| **听觉** | 声音、噪音、沉默、回响 | 营造空间感和情绪 |
| **嗅觉** | 气味、腐臭、花香、血腥味 | 最容易触发记忆和本能反应 |
| **触觉** | 温度、质感、疼痛、风 | 建立身体感受的沉浸感 |
| **味觉** | 苦涩、甘甜、腥咸 | 最少用但冲击力最强 |

**感官选择指南**:
- 战后场景 → 优先嗅觉（血腥）+ 触觉（伤痛）+ 视觉（废墟）
- 自然秘境 → 优先视觉（奇景）+ 听觉（异声）+ 嗅觉（灵气芬芳）
- 密室/地牢 → 优先触觉（阴冷）+ 嗅觉（霉味）+ 听觉（滴水/回声）
- 宴会/市集 → 优先视觉（繁华）+ 嗅觉（食物）+ 听觉（喧嚣）

**违反示例**: 整段场景描写只有"这里很美，到处都是花和树"（纯视觉+纯概括）。

**合规示例**: 灵花散发淡淡幽香（嗅觉），花瓣在灵气漩涡中缓缓旋转（视觉），远处瀑布轰鸣声被阵法削弱为轻柔水音（听觉），脚下灵草柔软如毯（触觉）。

## 规则 D-02：人物出场 — 标志性烙印

**约束**: 每个重要角色（有名字+超过 3 章戏份）首次出场时，必须在 **同一段落或连续段落** 中完成三项标志性描写：

1. **外貌特征** (1-2 句): 不是面面俱到的证件照式描写，而是 **1-2 个最具辨识度的特征**（伤疤/发色/眼神/体态/装饰）
2. **动作习惯** (1 句): 角色出场时的第一个动作，暗示性格（如"轻叩桌面"暗示沉思/掌控欲）
3. **语言特征** (1-2 句): 首次开口的语气/用词/口癖（如称呼方式、句式长短、敬语使用）

**优先级**: 当篇幅紧张时，按 语言 > 动作 > 外貌 的优先级保留。

**理由**: 读者对角色的第一印象决定了后续的代入感。三重烙印确保角色"活"起来。

**违反示例**: "这时，张三走了过来。他身高一米八，黑发黑眼，穿着白色长衫。"（纯外貌，无辨识度）

**合规示例**: 来人还没进门，一道慵懒的声音先飘了进来："我说怎么空气都变浊了，原来是你们这些废物在喘气。"（语言特征）一个身穿青衫的年轻人倚在门框上，眼尾微挑，带着几分玩世不恭的笑意（外貌+动作）。

## 规则 D-03：环境服务 — 描写必须为叙事服务

**约束**: 所有超过 200 字的环境描写必须明确服务于以下至少 **1 个叙事目标**：

1. **氛围营造**: 为即将发生的事件定调（如暴风雨前的压抑）
2. **情绪映射**: 外化角色内心状态（如阴雨天映射角色低落）
3. **伏笔埋设**: 环境细节暗示后续情节（如墙上的裂纹暗示阵法即将崩溃）
4. **对比强化**: 与前/后场景形成反差（如战前繁华vs战后废墟）
5. **角色衬托**: 环境反映角色地位/能力/变化（如居所描写暗示财力）

**判断标准**: 删除这段环境描写后，如果故事理解不受任何影响，则该描写是冗余的。

**违反示例**: 连续 300 字描写一座城市的历史沿革、建筑风格、街道布局，但故事中角色只是路过。

**合规示例**: 同样描写城市，但重点放在"城门口守卫森严到不正常"和"空气中弥漫着焦灼气息"，暗示即将发生的事件。

## 规则 D-04：比喻节制 — 避免辞藻堆砌

**约束**: 比喻使用频率不超过 **每 1000 字 2 个**，且需遵守以下原则：

1. **避免俗套比喻**: 以下比喻视为"禁用/慎用"：
   - "如同一头猛虎"（及所有猛兽比喻）
   - "宛如天仙"（及所有仙女/神仙比喻）
   - "好像被雷击中"（及所有天气比喻）
   - "仿佛时间静止"（及所有时间操控比喻）

2. **比喻来源多样性**: 连续 3 个比喻不能来自同一领域（如不能连续 3 个天气比喻）

3. **新颖度检查**: 每个比喻应至少包含 1 个"意外匹配点"——本体和喻体之间的关联不是读者立刻能想到的

**理由**: 比喻是调味料，不是主菜。过多比喻打断阅读节奏，俗套比喻降低文笔感知。

**违反示例**: "他的剑快如闪电，势如破竹，锋利得如同剃刀。"（连续 3 个常见比喻，无新意）

**合规示例**: "他的剑意像一根细针穿过喧闹的集市——周围所有人都安静下来，不是被震慑，而是本能地不想让针尖偏移。"（意外的匹配点：安静的原因不是恐惧而是珍惜）

## 规则 D-05：留白原则 — 关键场景的不写之写

**约束**: 在以下关键场景中，必须保留约 **20%** 的描写空白，交给读者自行补完：

1. **情感高潮**: 角色最脆弱/最幸福的瞬间，不完全写透，用省略号或动作暗示
2. **震撼场景**: 重大揭露/真相揭示，不逐字描写角色反应，留反应空间
3. **死亡场景**: 角色临终时，不过度描写痛苦细节，用一句话/一个意象收束

**实操方法**:
- 用动作代替心理描写（"他紧了紧拳头"代替"他内心翻涌着复杂的情感"）
- 用环境回应代替直接评价（"窗外的风停了"代替"空气凝固了"）
- 用省略暗示无法言说的部分（"她张了张嘴，最终什么都没说。"）

**理由**: 写满了反而乏味。读者自行补完的想象永远比文字更有力量。

**违反示例**: 角色死亡时，连续 500 字描写每寸伤势、每个器官的衰竭、每滴眼泪的轨迹。

**合规示例**: "她的手指慢慢松开，像是终于找到了一个可以安睡的姿势。风卷起她的发丝，遮住了那张不再有表情的脸。"

## 规则 D-06：信息分散释放 — 避免设定倾泻 (Info Dump)

**约束**: 世界观/设定/背景信息必须分散在 **3-5 个不同场景** 中逐步释放，而非一次倾泻：

**释放节奏**:

| 信息类型 | 建议释放方式 | 单次释放上限 |
|---------|-------------|-------------|
| 力量体系 | 通过实战/修炼场景展示 | 每次不超过 1 个层级 |
| 历史背景 | 通过对话/典籍/回忆插入 | 每次不超过 150 字 |
| 组织架构 | 通过角色互动展现 | 每次介绍不超过 2 个势力 |
| 技能设定 | 使用时才解释 | 每次不超过 1 个技能的机制 |

**核心原则**: **Show, Don't Tell** — 让读者通过角色的经历理解设定，而非通过旁白解释设定。

**违反示例**: 第 3 章用整章（3000 字）介绍修仙界的九大境界、三大宗门、灵石货币体系、丹药分级。

**合规示例**: 第 3 章通过主角在坊市购买物品时自然了解灵石价值；第 8 章通过与前辈切磋得知境界差距；第 15 章通过加入宗门了解宗门体系。

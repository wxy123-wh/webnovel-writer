# 言情/甜宠写作规则 (Romance Writing Rules)

> 以下规则由 `romance` profile 加载，适用于言情、甜宠、都市恋爱、古代言情等以感情线为核心的题材。

---

## 规则 R-01：感情线核心 — 绝对主线地位

**约束**: 感情进展是 **绝对主线**，事业线/剧情线必须为感情线服务，不能压制感情线：

### 主次关系

```
感情线 (主线) ═══════════════════════════════════►
  ↑ 服务     ↑ 服务     ↑ 服务     ↑ 服务
事业线 ─────► 剧情线 ─────► 配角线 ─────► 世界线
```

### 感情线保障

1. **篇幅保障**: 每 3 章中至少有 1 章以感情互动为核心内容
2. **非断档**: 连续不超过 **5 章** 没有任何感情线进展（哪怕是微进展）
3. **事业推动感情**: 事业上的事件应影响感情关系（升职→表白契机、危机→互相扶持）
4. **感情不中断**: 即使在事业高潮章节，也应穿插至少 1 个感情相关的细节

### 事业线处理原则

- 事业线是感情线的"催化剂"和"背景板"
- 事业的成功/失败应映射到感情关系的变化
- 如果一段事业剧情对感情线毫无影响，考虑是否需要这段剧情

**违反示例**: 连续 8 章都是商业博弈，男女主角之间的互动为零。

**合规示例**: 商业博弈中，男女主角被迫合作（事业线驱动）→ 合作中产生矛盾和默契（感情线进展）→ 商业成功后一起庆祝（感情线兑现）。

## 规则 R-02：心动瞬间 — 每章必有的甜蜜

**约束**: 每章至少包含 **1 个心动场景**，类型可从以下选择：

### 心动场景类型库

| 类型 | 描写重点 | 强度 |
|------|---------|------|
| **眼神交汇** | 对视时的微妙感觉 | ★☆☆ |
| **不经意触碰** | 手指相触、擦肩而过 | ★★☆ |
| **保护性动作** | 挡在身前、拉住手腕、披外套 | ★★★ |
| **宠溺对白** | 带着爱意的调侃或关心 | ★★☆ |
| **吃醋表现** | 因为对方与他人亲近而不悦 | ★★★ |
| **温柔细节** | 记住对方的习惯/喜好 | ★★☆ |
| **突然的亲密** | 意外的拥抱、壁咚、额头贴 | ★★★ |
| **心疼瞬间** | 看到对方脆弱时的心疼 | ★★★ |
| **心动独白** | 内心意识到自己动心的瞬间 | ★★☆ |
| **旁人助攻** | 他人点破两人之间的暧昧 | ★★☆ |

### 心动场景写作原则

1. **不重复**: 连续 3 章不能使用同一类型的心动场景
2. **有铺垫**: 心动场景前应有情境铺垫，不能突兀
3. **有反应**: 心动发生后，角色的反应比心动本身更重要（红了耳尖/心跳加速/假装不在意）
4. **有递进**: 心动场景的亲密程度应随关系进展递增

**违反示例**: 连续 5 章的心动场景都是"他看了她一眼，她脸红了"。

**合规示例**: 第 1 章眼神交汇（★）→ 第 2 章不经意碰触（★★）→ 第 3 章保护性动作（★★★）→ 第 4 章宠溺对白（★★）→ 第 5 章吃醋表现（★★★）——类型递进。

## 规则 R-03：甜虐节奏 — 7:3 黄金比例

**约束**: 甜（甜蜜/心动/暖心）与虐（误会/分离/心痛）的比例建议保持在 **7:3**：

### 节奏模式

```
甜 → 甜 → 甜 → 微虐 → 甜 → 虐 → 甜（糖来补偿）→ 甜 → 甜 → 虐 → 甜（补偿）
```

### 虐的设计原则

1. **虐必有因**: 不能无理由地制造矛盾（误会/分离必须有合理的情境触发）
2. **虐必有糖**: 每段虐后必须在 **2 章内** 给予补偿（甜蜜/和好/解释）
3. **虐不伤害核心**: 虐不能伤害感情线的根本（如不能出现不可原谅的背叛）
4. **虐的层次**: 虐的程度可以递增，但每次都要有对应的更强的"糖"来补偿

### 虐的类型与对应糖

| 虐的类型 | 强度 | 对应的糖 |
|---------|------|---------|
| 小误会 | ★☆ | 解释+甜蜜互动 |
| 被迫分离 | ★★ | 重逢场景+加倍甜蜜 |
| 第三者干扰 | ★★ | 吃醋+明确表态 |
| 家族/身份阻力 | ★★★ | 为对方打破规则/放弃一切 |
| 生死危机 | ★★★ | 守候/表白/劫后余生 |

**违反示例**: 连续 5 章都是虐（误会→分离→第三者→家族反对→误会加深），读者情绪崩溃。

**合规示例**: 甜蜜 3 章 → 小误会（1 章）→ 解释+更甜蜜（2 章）→ 被迫分离（1 章）→ 重逢+表白（2 章）→ 甜宠日常（2 章）→ 家族阻力出现（1 章）→ 携手面对+突破（2 章）。

## 规则 R-04：身份差 — 利用差距制造张力

**约束**: 利用角色之间的 **身份差距** 制造张力和戏剧冲突：

### 身份差类型

| 差距类型 | 典型设定 | 制造的冲突 |
|---------|---------|-----------|
| **门第差距** | 豪门vs平民 | 家族反对、社交圈排斥 |
| **财富差距** | 富可敌国vs穷困潦倒 | 生活方式冲突、自卑感 |
| **地位差距** | 总裁vs实习生、王爷vs丫鬟 | 权力不对等、身份揭露 |
| **年龄差距** | 成熟vs年轻 | 代沟、保护欲vs独立 |
| **种族/身份** | 人类vs非人类（仙凡恋等） | 跨越规则的代价 |
| **敌对身份** | 商业对手/家族仇人 | 禁忌之恋的刺激感 |

### 身份差写作原则

1. **差距要显现**: 差距不能只存在于设定中，要体现在具体情节（如宴会上格格不入、被对方家人看不起）
2. **双方都有成长**: 缩小差距应是双向的（强者学会尊重/弱者获得自信），不是一方单方面改变
3. **差距是动力**: 差距应推动情节发展，不能只是背景设定
4. **跨越有代价**: 打破身份差的过程不应太容易，要有挣扎和代价

**违反示例**: 设定了"豪门vs平民"的身份差，但豪门家人立刻就接受了，没有任何阻力。

**合规示例**: 平民女主在豪门宴会上被嘲笑→男主出面维护但引发家族压力→女主决定用自己的能力证明价值→男主顶住家族压力坚持选择→最终女主的事业成功让家族改观——差距是一步步缩小的。

## 规则 R-05：吃醋桥段 — 关系升温催化剂

**约束**: 适度使用吃醋/误会让关系升温，但需注意频率和方式：

### 使用频率

- 每 **5-8 章** 可安排 1 次吃醋桥段
- 连续 2 次吃醋的原因不能相同
- 吃醋后的关系升温必须比吃醋前更进一步

### 吃醋桥段设计

| 阶段 | 内容 | 篇幅 |
|------|------|------|
| **触发** | 出现让角色嫉妒的情境/人物 | 0.5 章 |
| **表现** | 角色的吃醋反应（言语/行为/情绪） | 0.3 章 |
| **暴露** | 嫉妒情绪被对方或旁人发现 | 0.2 章 |
| **升温** | 因为吃醋事件关系更进一步 | 0.5 章 |

### 吃醋表现方式

1. **言语型**: 说话带刺、阴阳怪气、有意无意提到对方
2. **行为型**: 故意冷落、反常地热情、跟踪/查岗
3. **情绪型**: 闷闷不乐、容易发脾气、注意力不集中
4. **反差型**: 平时冷静的人突然不理智了——反差本身就是心动证据

### 禁止类型

- ❌ 因为吃醋对伴侣使用暴力/冷暴力
- ❌ 吃醋后不沟通导致严重误会（信息不对称式虐）
- ❌ 同样的吃醋桥段重复使用（每次都是"前任出现→吃醋→和好"）

**违反示例**: 每 2 章就来一个情敌，每次都是男主吃醋→霸道宣言→和好，完全相同的模式。

**合规示例**: 第一次：男主因为女主和同事走得近而不高兴（言语型吃醋→确认感情）；第二次：女主因为男主的前任出现而不安（情绪型吃醋→交心）；第三次：双方同时为对方吃醋（双线吃醋→共同承认感情）。

## 规则 R-06：表白时机 — 最佳窗口期

**约束**: 正式表白不能发生在前 **30%**（感情基础未建立），也不能拖到后 **10%**（太晚缺少回味空间）：

### 表白时机设计

| 阶段 | 位置 | 说明 |
|------|------|------|
| **暗示期** | 10%-30% | 双方有暧昧但未明确，读者在猜测 |
| **最佳表白窗** | 30%-70% | 感情基础已建立，正式表白推进关系 |
| **确认期** | 70%-90% | 表白后的磨合与深化 |
| **稳定期** | 90%-100% | 感情稳定的收束与升华 |

### 表白设计要素

1. **有铺垫**: 表白前应有足够的感情积累（至少 3 个心动场景+1 次深度互动）
2. **有仪式感**: 表白场景需要有特殊感（特殊的地点/时刻/方式）
3. **有紧张感**: 表白前的紧张和不确定是重要的情感体验
4. **有回应**: 对方的回应同样重要（接受/犹豫/拒绝→后续发展）

### 表白方式参考

- **直球型**: 直接说出口（适合外向角色/高压情境）
- **行动型**: 用行动表白（准备惊喜/做出选择/保护到底）
- **间接型**: 通过第三方/书信/暗示传达（适合害羞/身份受限角色）
- **情境型**: 在特殊情境下情不自禁（危险后/醉酒后/失控后）

**违反示例**: 第 2 章就表白（读者还不了解角色、感情毫无基础），或第 98 章才表白（没有回味空间）。

**合规示例**: 第 5 章心动 → 第 12 章暧昧 → 第 18 章吃醋确认感情 → 第 25 章（约40%处）在雨天告白 → 第 26-80 章确认关系后的甜蜜+磨合 → 结局婚礼/承诺。

## 规则 R-07：关系进展标记 — 阶段性事件

**约束**: 感情关系的每个阶段必须有 **明确的标志性事件** 作为进展标记：

### 关系阶段与标记

| 阶段 | 标志性事件（至少1个） | 读者感受 |
|------|---------------------|---------|
| **陌生人→初识** | 第一次正式对话/第一次注意到对方 | "他们要认识了吗？" |
| **初识→暧昧** | 第一次心跳加速/第一次超出普通朋友的关心 | "他/她动心了！" |
| **暧昧→确认** | 正式表白/第一次牵手/公开关系 | "终于在一起了！" |
| **确认→磨合** | 第一次矛盾/第一次为对方改变/第一次见家长 | "在一起后也不容易" |
| **磨合→稳定** | 共同克服重大困难/对方的承诺/双方家庭认可 | "他们真的很配" |
| **稳定→升华** | 共同面对生死考验/为对方牺牲/最终承诺 | "这就是爱情" |

### 进展保障

1. **每阶段必标记**: 不能跳过任何阶段（确认关系前必须有暧昧期）
2. **标记可追溯**: 读者能明确说出"他们是第X章确认关系的"
3. **速度可控**: 每个阶段至少停留 3-5 章（不能一见面就确认关系）
4. **回溯可见**: 从后往前看，每个阶段的进展都有清晰的事件支撑

### 阶段转换信号

- **语言变化**: 称呼改变（"喂"→名字→昵称）
- **行为变化**: 距离感改变（并排走→牵手→拥抱）
- **心理变化**: 独白内容改变（"这个人真烦"→"他/她其实还不错"→"完了，我好像……"）

**违反示例**: 前 5 章还是陌生人，第 6 章突然就在一起了，跳过了暧昧期，缺少心动过程。

**合规示例**: 第 1-3 章初识（矛盾初遇/被迫合作）→ 第 4-8 章逐渐了解（发现对方的另一面/首次心动）→ 第 9-15 章暧昧期（若即若离/吃醋/暗示）→ 第 16-20 章表白确认→ 第 21 章起确认后的甜蜜+磨合。每一步都有明确的事件标记。

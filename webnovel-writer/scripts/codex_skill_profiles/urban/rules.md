# 都市异能写作规则 (Urban Power Writing Rules)

> 以下规则由 `urban` profile 加载，适用于都市异能、现代修真、隐世高手等以现代背景为舞台的题材。

---

## 规则 U-01：身份隐藏 — 掉马节奏

**约束**: 异能者必须在普通社会中隐藏身份，身份暴露（"掉马"）是核心爽点，需要精细的节奏控制：

### 掉马节奏设计

| 阶段 | 位置 | 内容 |
|------|------|------|
| **深藏不露** | 前 20% | 主角以普通人身份活动，读者知道但书中角色不知道 |
| **微露端倪** | 20%-40% | 少数人开始怀疑，主角用更高明的手段掩饰 |
| **部分掉马** | 40%-60% | 核心圈子知道主角身份，但社会层面仍然隐藏 |
| **大掉马** | 60%-80% | 关键身份在重大事件中暴露，引发各方反应 |
| **身份重构** | 80%-100% | 身份公开后的新格局和新矛盾 |

### 掉马爽点设计

1. **被迫暴露**: 为保护重要的人/事不得不使用能力
2. **无意暴露**: 能力失控或被意外拍到/目击
3. **主动亮明**: 在装逼打脸场合，不再隐藏实力
4. **层层揭开**: 不是一次性全部暴露，而是"掉一层还有一层"

**违反示例**: 主角第 5 章就在大庭广众下使用异能，之后再也不隐藏，掉马的紧张感消失。

**合规示例**: 前 30 章主角在同事面前只是"体能好一点的普通人"——直到公司年会遇到恐怖袭击，他不得不在所有人面前挡下一颗子弹。

## 规则 U-02：现实锚定 — 社会结构的合理性

**约束**: 即使存在异能设定，社会结构、经济体系、法律制度仍需基本合理：

### 现实锚定清单

| 领域 | 要求 |
|------|------|
| **经济** | 有收入来源和消费逻辑，不能凭空有钱 |
| **法律** | 异能者犯法也有后果（官方组织/暗杀名单） |
| **社会** | 普通人的反应要真实（恐惧/崇拜/排斥） |
| **媒体** | 信息时代，大事件不可能完全掩盖 |
| **组织** | 异能组织有结构/层级/规矩，不是无政府状态 |

### 常见错误规避

- ❌ 主角住在豪宅却从不提收入来源
- ❌ 异能者随意杀人无人追究
- ❌ 大规模战斗后新闻毫无报道
- ❌ 异能组织的运作经费不明

**违反示例**: 主角是普通上班族，但住别墅开豪车，经济逻辑完全断裂。

**合规示例**: 主角表面是普通上班族，暗地通过地下渠道接异能委托赚钱，购房用假名+现金，解释了经济来源同时增加神秘感。

## 规则 U-03：三章一峰 — 紧凑节奏模式

**约束**: 都市异能建议采用 **"三章一峰"** 的基础节奏模式：

### 节奏模板

```
第1章: 困境/挑战出现
  → 主角遭遇麻烦（职场/人际关系/异能世界）
  → 问题看似无法用普通手段解决
  → 章末：读者期待主角如何应对

第2章: 能力初展/布局
  → 主角暗中开始行动
  → 展示部分能力但不完全暴露
  → 章末：局面开始朝有利方向变化

第3章: 小胜+新阻力
  → 困境得到阶段性解决（爽点兑现）
  → 但胜利引出新的问题或更大的对手
  → 章末：新的悬念（更大的挑战在前方）
```

### 弹性调整

- 小矛盾：压缩为 2 章解决
- 大高潮：扩展为 5-6 章，但保持"困境→行动→结果"的三段结构
- 过渡章：用于展示能力后果、关系推进，但不超过 2 章连续

**违反示例**: 困境出现后拖了 8 章才解决，中间充斥日常描写，读者失去耐心。

**合规示例**: 第 11 章遭遇商业对手打压 → 第 12 章暗中调查+异能辅助 → 第 13 章反击成功但发现背后有更大的势力。

## 规则 U-04：装逼打脸 — 核心爽点多样化

**约束**: 装逼打脸是都市异能的核心爽点模式，但不能每次都用同样的方式：

### 打脸模式库

| 模式 | 说明 | 使用频率建议 |
|------|------|-------------|
| **实力碾压** | 纯粹的武力/能力压制 | 每 10 章限 1 次 |
| **知识碾压** | 在专业领域展现远超常人的见识 | 每 8 章限 1 次 |
| **身份碾压** | 真实身份/背景曝光让对方傻眼 | 每 15 章限 1 次 |
| **人脉碾压** | 关键人物出场为主角站台 | 每 12 章限 1 次 |
| **反转碾压** | 被人看不起后用结果说话 | 每 8 章限 1 次 |
| **迂回打脸** | 不直接冲突，让对方自己发现差距 | 每 10 章限 1 次 |

### 打脸设计原则

1. **有铺垫**: 对手的嚣张要有铺垫，否则打脸无快感
2. **有过程**: 不是瞬间碾压，要有"对方从嚣张到震惊到怀疑人生"的过程
3. **有代价**: 打脸后可能引来更强的对手或官方关注
4. **有变化**: 连续 3 次打脸不能使用同一种模式

**违反示例**: 每次冲突都是：对手嘲讽 → 主角出手 → 对手被秒杀 → 旁观者震惊。模式完全重复。

**合规示例**: 第一次实力碾压（打倒对方保镖）；第二次知识碾压（在专业领域让对方出丑）；第三次人脉碾压（对方想用关系压人，结果发现主角认识的人更大）。

## 规则 U-05：产业链博弈 — 行业基本常识

**约束**: 当故事涉及具体行业（房地产/金融/娱乐/餐饮/科技）时，必须具备基本常识：

### 行业常识清单

| 行业 | 必须了解的基础 |
|------|---------------|
| **房地产** | 拿地流程、开发周期、资金链 |
| **金融** | 基本金融产品、监管规则 |
| **娱乐圈** | 经纪体系、资本运作、舆论公关 |
| **餐饮** | 选址、供应链、人效坪效 |
| **科技** | 研发周期、技术壁垒、专利布局 |
| **医疗** | 基本诊疗流程、医患关系 |

### 写作原则

1. **不装懂**: 不确定的细节用模糊化处理，不要写具体但错误的数字
2. **用角色**: 通过行业角色之口展示专业知识（让内行角色替作者说话）
3. **重逻辑**: 不需要完全正确，但逻辑链必须自洽
4. **避硬伤**: 绝对不能出现行业常识性错误（如"公司上市只需一周"）

**违反示例**: 主角投资 100 万，一个月后回报 10 亿，无任何合理解释。

**合规示例**: 主角利用异能预判市场趋势（有超自然依据），配合精准的时间点操作（有专业逻辑），在三个月内获得百倍回报（时间合理化）。

## 规则 U-06：异能代价 — 能力的限制

**约束**: 使用异能必须有明确的 **代价/冷却/限制**，不能无限使用：

### 代价设计模板

| 代价类型 | 说明 | 示例 |
|---------|------|------|
| **体力消耗** | 使用后身体疲惫，需休息恢复 | 使用念动力后手臂酸软 |
| **精神负担** | 对精神/意识的影响 | 读心术后头痛欲裂 |
| **冷却时间** | 能力使用后的强制等待期 | 瞬移后 24 小时内不能再使用 |
| **累积伤害** | 频繁使用的累积负面效果 | 每次超速再生缩短寿命 |
| **暴露风险** | 使用越强越容易被追踪 | 释放高阶能力会暴露位置 |
| **道德代价** | 对使用者心理的影响 | 操控他人后产生愧疚感 |

### 代价设计原则

1. **明确前宣**: 代价规则在能力首次展示时就应暗示或说明
2. **逐步揭示**: 代价可以随使用加深而逐步显现（但一旦确立不可矛盾）
3. **影响剧情**: 代价要在关键时刻发挥作用（不能只是嘴上说有代价却从不受影响）
4. **不能消除**: 除非付出更大的代价来减轻（不能轻松解决代价问题）

**违反示例**: 主角的能力是"时间停止"，但可以无限次使用且无任何副作用。

**合规示例**: 主角可以停止时间 3 秒，每次使用后必须等待 1 小时冷却，且频繁使用会加速细胞老化。在关键战役中，他不得不连续使用导致头发瞬间花白——代价具象化。

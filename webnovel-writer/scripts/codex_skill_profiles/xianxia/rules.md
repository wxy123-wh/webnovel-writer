# 修仙/玄幻写作规则 (Xianxia Writing Rules)

> 以下规则由 `xianxia` profile 加载，适用于修仙、玄幻、仙侠等以修炼体系为核心的题材。

---

## 规则 X-01：境界体系 — 突破的仪式感

**约束**: 设定 **8-10 级** 的境界体系，每次突破必须满足以下条件：

### 境界设计要求

| 要素 | 要求 |
|------|------|
| 层级数量 | 8-10 级（推荐 9 级，呼应"九九归一"） |
| 层级命名 | 有文化内涵，避免纯数字（如炼气→筑基→金丹→元婴……） |
| 层级差距 | 每级之间的实力差距要明确且递增（越高级差距越大） |
| 突破难度 | 每级突破难度递增，不能每次都"水到渠成" |

### 突破仪式感要求

每次境界突破（至少是主角的突破）必须包含以下 **3 个阶段**：

1. **瓶颈期** (1-3 章): 描写突破前的困境——理解不了道、找不到契机、反复失败
2. **突破过程** (1 章): 不能一笔带过，必须有至少 1 段修炼感悟/天地异象/身体变化的描写
3. **突破后变化** (1-2 章): 展示突破后的实力变化，让读者感受到"升级"的快感

### 突破代价

每次突破必须伴随某种代价（不能零成本升级）：

| 代价类型 | 示例 |
|---------|------|
| 资源消耗 | 灵石/丹药/天材地宝的大量消耗 |
| 时间成本 | 闭关数月/数年，期间可能错过重要事件 |
| 身体负担 | 经脉扩张的痛苦、重塑肉身的危险 |
| 机缘依赖 | 需要特定的天时/地利/人和 |
| 心魔考验 | 内心的执念/恐惧考验 |

**违反示例**: "三天后，他突破了。"（一笔带过，无过程、无代价、无仪式感）

**合规示例**: 用整整一章描写：瓶颈的焦躁→偶然的顿悟→引动天地灵气→身体剧痛如万蚁噬骨→丹田中金丹凝聚成型→突破后的玄妙感受→实力的质的飞跃。

## 规则 X-02：资源货币化 — 核心微兑现载体

**约束**: 灵石/丹药/功法是修仙世界的"货币体系"，必须建立清晰的价值尺度：

### 资源层级

| 资源类型 | 功能 | 获取方式 |
|---------|------|---------|
| 灵石 | 通用货币、修炼辅助 | 任务/交易/挖矿 |
| 丹药 | 辅助突破/疗伤/增强 | 炼制/购买/抢夺 |
| 功法 | 修炼方法/战斗技能 | 传承/自创/夺获 |
| 法宝 | 战斗装备/辅助工具 | 炼制/遗物/奖励 |
| 天材地宝 | 稀有资源/突破关键 | 探索/机缘/拍卖 |

### 价值尺度规则

1. **等价交换**: 确定一个基准单位（如下品灵石），所有资源有相对明确的价值比
2. **获得即兑现**: 主角获得资源后，必须在 3-5 章内"消费"（使用/消耗/交易），形成兑现闭环
3. **资源压力**: 修炼资源不能无限充裕，需要持续的资源获取压力

**违反示例**: 灵石在前 20 章是稀缺资源，第 21 章后主角突然再也不缺灵石了（无合理解释）。

**合规示例**: 灵石始终是重要资源，主角通过不同方式持续获取（任务→拍卖→炼丹售卖→开辟灵矿），但支出也在增加（修炼→购买材料→打点关系），形成动态平衡。

## 规则 X-03：逆天改命 — 合理的突破常规

**约束**: 主角可以超越常规、打破规则，但每一次"逆天"行为都必须有合理的解释基础：

### 合理解释类型

| 解释类型 | 说明 | 使用限制 |
|---------|------|---------|
| **机缘** | 偶然获得的上古传承/仙人遗物 | 每个大境界限 1 次，需要铺垫 |
| **悟性** | 对道/法则的超常理解能力 | 需要前期铺垫悟性表现 |
| **特殊体质** | 天生体质带来的优势 | 开篇设定，中后期逐步解锁 |
| **外力辅助** | 法宝/丹药/阵法的辅助 | 有副作用或使用限制 |
| **以命相搏** | 极端情况下燃烧寿命/精血 | 有不可逆的代价 |

### 禁止类型

- 无缘无故的天赋碾压（"因为他就是天才"）
- 重复使用同一种解释（不能每次都靠"悟性超群"）
- 没有代价的逆天行为

**违反示例**: 主角每次越级战斗都"凭一股不屈的意志"获胜，无其他解释。

**合规示例**: 第一次越级靠体内隐藏的上古血脉觉醒（有铺垫）；第二次靠战前准备的阵法陷阱（有智慧）；第三次靠燃烧精血的禁术（有代价）——方式多样，各有依据。

## 规则 X-04：宗门结构 — 正邪之间的灰色地带

**约束**: 宗门/势力结构不能是简单的"正派=好人，邪派=坏人"，必须包含：

### 势力设计要素

1. **正邪对立**: 存在明确的正道/邪道阵营区分
2. **灰色地带**: 正道中有伪君子，邪道中有真性情
3. **立场转换**: 至少 1 个重要角色在正邪之间转换立场（需要充分动机）
4. **利益驱动**: 势力冲突的根本原因是利益/资源/理念，而非纯粹的善恶

### 宗门互动模式

- **表面和谐**: 正道宗门之间的竞争与合作
- **暗流涌动**: 宗门内部的派系斗争
- **利益交换**: 宗门间的联姻/交易/结盟
- **信仰分歧**: 对"道"的不同理解导致的对立

**违反示例**: 所有正道宗门成员都是好人，所有邪道宗门成员都是坏人，无任何例外。

**合规示例**: 正道魁首宗门中，长老为了宗门利益可以牺牲弟子；邪道宗门中，有人是因为被正道陷害而投靠，对同伴重情重义。

## 规则 X-05：天道法则 — 高阶约束

**约束**: 高阶修士受天道约束，不能随意出手干预低阶事务：

### 天道约束机制

1. **因果**: 高阶修士对低阶出手会招致天劫/因果反噬
2. **境界封锁**: 某些区域限制高阶修士进入（如秘境的境界上限）
3. **天劫威胁**: 每次大境界突破都有天劫，频繁出手会增加天劫威力
4. **寿元限制**: 高阶不等于永生，突破失败会折损寿元

### 约束的意义

- 为主角创造成长空间（如果没有天道约束，高阶修士随时可以碾压主角）
- 为矛盾冲突设定天花板（不能无限升级）
- 为"破局"创造戏剧性（打破天道约束本身就是一个高潮点）

**违反示例**: 化神期老祖随时出手干预炼气期小辈的争斗，且无任何代价。

**合规示例**: 化神期老祖想出手，但感知到天劫将至，不敢轻举妄动；或出手后发现因果缠身，加速了天劫降临。

## 规则 X-06：机缘设计 — 四步走模式

**约束**: 每个重大机缘必须包含完整的 **四步流程**：

### 四步流程

| 步骤 | 内容 | 建议篇幅 |
|------|------|---------|
| **1. 发现** | 通过探索/线索/意外发现机缘存在 | 1-2 章 |
| **2. 获取** | 克服考验/竞争/谜题获取机缘 | 2-3 章 |
| **3. 消化** | 理解/炼化/适应机缘的力量 | 1-2 章 |
| **4. 升级** | 机缘带来的实质性提升 | 1 章 |

### 机缘设计要点

1. **发现不能太容易**: 必须有前置条件（特定修为/特殊体质/机缘巧合）
2. **获取必须有考验**: 不是白送，需要付出代价或通过考验
3. **消化不能跳过**: 至少用 1 段描写主角消化机缘的过程和感悟
4. **升级要有仪式感**: 展示机缘带来的具体变化（数值/能力/体质）

**违反示例**: 主角路过一个山洞，捡到一本功法，翻两页就学会了，立即战力翻倍。

**合规示例**: 主角注意到山壁上的异常纹路（发现）→ 解开阵法谜题进入洞府（获取）→ 闭关参悟功法，经历多次失败和顿悟（消化）→ 成功修炼后功法与体质融合，实力产生质变（升级）。

## 规则 X-07：修炼过程 — 不可跳过的感悟

**约束**: 修炼/参悟过程不能一笔带过，至少包含 **1 段** 具体的修炼感悟描写：

### 感悟描写要素

每次重要的修炼/突破场景，应包含以下至少 2 项：

1. **道的理解**: 对某种法则/道理的个人理解（用角色的语言表达）
2. **身体感受**: 灵气运转时的身体感受（温热/刺痛/通透）
3. **意象描写**: 修炼中看到的幻象/景象（如身处星空/海底/烈焰中）
4. **对比感悟**: 与之前理解的对比（"原来如此"的领悟时刻）
5. **关联感悟**: 将修炼感悟与人生经历联系（道与生活的共鸣）

### 感悟写作风格建议

- 避免纯理论说教，用比喻和画面感传达
- 感悟应与角色当前的人生阶段匹配（初入修行的感悟≠经历生死后的感悟）
- 可以用"一段感悟+一个具体变化"的结构，让抽象道理落地

**违反示例**: "他盘膝而坐，运功一个时辰后，突破到了筑基期。"

**合规示例**: "灵气如溪水般涌入经脉，他感觉自己像是一条干涸已久的河流终于迎来了雨季。之前所有的修炼都是在河床上挖渠，而此刻，水来了。他的意识沉入丹田，看见那枚虚幻的灵气漩涡开始凝实——不再是雾，而是液态的灵力。筑基，就是在丹田中筑起第一道堤坝。"
